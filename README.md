# 🔐 Secure Login System

A secure and responsive web-based login system developed as part of my cybersecurity internship. The project focuses on implementing secure authentication practices such as password hashing, input validation, and session management to protect user accounts from common attacks.

---

## 📌 Features

- ✅ User Registration
- ✅ Secure User Login
- ✅ Password Hashing using bcrypt
- ✅ Input Validation
- ✅ SQL Injection Protection
- ✅ Session Management
- ✅ Logout Functionality
- ✅ Responsive User Interface

## 🔒 Security Features

- Passwords are securely hashed before storing.
- User input is validated and sanitized.
- Prevents SQL Injection attacks.
- Secure session handling.
- Logout destroys active sessions.
- Protected routes for authenticated users.

## 🚀 Installation

Clone the repository

```bash
git clone https://github.com/Divyanshneekhra/Secure-Login-System.git
```

Move into the project directory

```bash
cd Secure-Login-System
```

Install dependencies

```bash
npm install
```

Run the application

```bash
npm start
```

Open your browser

```
http://localhost:3000
```
